# Editeur pour MarkDown
Pour l'editeur MarkDown, on peut utiliser: [ATOM Text Editor](https://atom.io/)

## Recuperation par PPA Ubuntu 64bits:
```
sudo add-apt-repository ppa:webupd8team/atom
sudo apt-get update
sudo apt-get install atom
```
